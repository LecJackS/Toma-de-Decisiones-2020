Ejercicio 4

Simular una ROC emprica.

Considerar un participante de un experimento de deteccin con \(d' = 1.5\).

En cada trial, el experimentador presenta "ruido" o "ruido+seal" con igual probabilidad.

La tarea del participante es responder "si" cuando la seal est presente y "no" cuando est ausente.

Considere que el participante adopta un criterio no sesgado y que la distribucin de ruido est centrada en cero.

A.

Simular (como en el ejercicio 7 de la Gua 2), 10000 trials del experimento.

Hacer un histograma de la seal interna correspondiente a cada estmulo.

dp <- 1.5
muN <- 0
muS <- muN + dp

sigma <- 1
# prop=0.2: 20% ruido, 80% seal
prop <- 0.5
ntrials <- 10000
ntrialsN <- ntrials*prop
ntrialsS <- ntrials*(1-prop)

# La informacin de qu estmulo corresponde a cada trial la vamos a guardar en la
# variable Label:
# seal+ruido ---> 1
# ruido ---> 0
# Crear un vector de Labels para cada tipo de estmulo. El largo de cada vector debe ser
# ntrialsS y ntrialsN
signalLabels <- rep(1, ntrialsS)
noiseLabels  <- rep(0, ntrialsN)

# Tomar las muestras de cada distribucin.
signalSamples <- rnorm(ntrialsS, mean=muS, sd=sigma)
noiseSamples  <- rnorm(ntrialsN, mean=muN, sd=sigma)

# Poner todas las muestras juntas, primero las de'seal+ruido', luego las de'ruido'
allSamples <- append(signalSamples, noiseSamples)

# Hacer lo mismo con los vectores de Labels
allLabels <- append(signalLabels, noiseLabels)
hist(allSamples, col=rgb(1,1,1, 0.5), prob=F, main="Histogramas de las muestras", yaxt="n")
grid(11)
m <- max(hist(allSamples, plot=F)$counts)
ytick<-seq(0, m, length.out=11)
axis(side=2, at=ytick, labels = FALSE)
text(par("usr")[1], ytick,  
     labels = round(ytick/m,2), pos = 2, xpd = TRUE)
hist(signalSamples, prob=F, add=T, col=rgb(0,0,1, 0.5))
hist(noiseSamples, prob=F, add=T, col=rgb(1,0,0, 0.5))

En el histograma anterior observamos ambas distribuciones estimadas superpuestas, con valores de seal a la derecha, y de ruido a la izquierda.

El participante solo observa una mezcla de ambas distribuciones, como se ve en blanco en el fondo.
